# Icalia Labs Services Deck

Interactive presentation showcasing Icalia Labs' **Nearshore Staff Augmentation** 
and **Agentic Engineering** services.

**Live:** https://icalialabs.github.io/icalia-labs-services-deck/

## Navigation
- **Arrow Keys** or **Space** — next slide
- **Left Arrow** — previous slide
- **Touch/Swipe** — navigate on mobile

## Building
No build step required. Push to `main` branch and GitHub Pages auto-deploys.
